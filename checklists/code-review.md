# Code Review Validation Checklist

- [ ] Story file loaded and status verified as "review"
- [ ] Epic and story IDs resolved
- [ ] Architecture/standards docs loaded
- [ ] Tech stack detected and documented
- [ ] Acceptance criteria cross-checked against implementation
- [ ] File List reviewed for completeness
- [ ] Tests identified and mapped to ACs; gaps noted
- [ ] Code quality review performed on all changed files
- [ ] Security review performed on changed files and dependencies
- [ ] Outcome decided (Approve / Changes Requested / Blocked)
- [ ] Review notes appended under "Senior Developer Review (AI)"
- [ ] Change Log updated with review entry
- [ ] Story status updated (done or in-progress)
- [ ] Sprint status synced
